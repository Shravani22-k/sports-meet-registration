# 🏆 Sports Meet Registration System

A full-stack web project for sports event registration.  
Users can register for sports, get a unique bib number, and view confirmation details.

## ✨ Features

- Sports registration form
- PHP backend
- MySQL database connection
- Unique bib number generation
- Confirmation page
- Admin page to view all registrations

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript
- PHP
- MySQL
- Git & GitHub

## 📁 Project Structure

```text
sports-meet-registration/
├── index.html
├── config.php
├── register.php

🗄️ Database Setup
Open MySQL Workbench
Run:
CREATE DATABASE sports_meet;
USE sports_meet;
Import or run database.sql

▶️ How to Run
php -S localhost:8000

Open:
http://localhost:8000/index.html

👀 View Records
http://localhost:8000/view_records.php

📌 Note
Update config.php with your own MySQL username and password before running.
