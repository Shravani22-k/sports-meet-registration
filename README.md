# 🏆 Sports Meet Registration System

A full-stack web project for sports event registration.  
Users can register for sports, get a unique bib number, and view confirmation details.

## ✨ Features

- Sports registration form
- PHP backend
- MySQL database connection
- Unique bib number generation
- Confirmation page
- Admin page to view all registrations

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript
- PHP
- MySQL
- Git & GitHub
